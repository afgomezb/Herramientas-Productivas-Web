Proceso de trabajo

En este espacio se desea mostrar el proceso que sigo para el desarrollo de un proyecto web, desde su inicio hasta su fin.

Visión General: Para iniciar el proyecto, se debe definir claramente una visión general de este, definir objetivos, alcance, involucrados en el proyecto, especificaciones tecnicas y tiempos de entrega.

Definición de la estructura: En esta etapa se tiene contacto directo con el cliente, con el fin de tomar los resultados que el cliente quiere sobre el diseño de la página, si dicho cliente no tiene bien claro su diseño, le ayudo a definirlos de manera concreta y directa. Una vez definidos estos aspectos, doy opciones contando con mi experiencia para conseguir una buenas estructura.

Diseño Visual: En esta etapa se realizan bosquejos teniendo en cuenta los contenidos que el cliente desea mostrar en la página web, para la eleccion de un diseño final.

Desarrollo del sitio web: aqui ya se tienen las bases para el desarrollo del sitio, se integra los aspectos ya definidos para asi contruir el sitio web.
Cuando se tiene un avance considerable, se hace una revisión con el cliente, con el fin de corregir lo que sea necesario en cuanto a contenido.

Pruebas: Luego de tener el sitio ya construido, se realizan pruebas diferentes pruebas para garantizar la funcionalidad, seguridad y usabilidad del sitio web. Despliegue: En la ultima etapa del proceso, se realiza capacitación y publicación del sitio web, se ejecuta el proyecto en un servidor con el fin de culminar su funcionamiento.
